# 6643_M31

## 業務簡介
**板塊:** Technology
**產業:** Semiconductors
**市值:** 23,863 百萬台幣
**企業價值:** 23,029 百萬台幣

M31 技術股份有限公司 (6643，円星科技) 為全球領先的半導體[[矽智財]] ([[Silicon IP]]) 設計與授權公司，是台灣矽智財產業的旗艦企業。核心產品涵蓋高速介面 IP (含 [[USB]] 2.0/3.x、[[PCIe]] Gen 3/4/5、[[MIPI]] D-PHY/C-PHY/M-PHY、[[SerDes]] 及 [[DDR]] 4/5)、基礎元件 IP (標準元件庫、[[SRAM]] Compiler、GPIO) 及高精度類比 IP。全球已超過 150 家 IC 設計公司採用 M31 的 IP，客戶涵蓋美國、日本、歐洲、韓國及台灣等地。M31 榮獲[[台積電]]年度最佳特殊製程 IP 夥伴獎，並持續開發針對先進製程節點 (3nm/2nm) 的 [[PCIe]] Gen 5/6 及 [[USB4]] 高速介面 IP，搶占 [[AI 伺服器]]及[[資料中心]]互連 IP 商機。

## 供應鏈位置
**上游 (EDA 工具與晶圓廠製程 PDK):**
- **EDA 軟體:** [[新思科技]] (Synopsys)、[[Cadence]] (IC 電路設計與驗證工具)
- **製程 PDK:** [[台積電]]、[[聯華電子]] 等晶圓廠提供設計規則及製程設計套件

**中游 (矽智財設計與授權):**
- **M31 技術 (円星科技):** [[USB]]、[[PCIe]]、[[MIPI]]、[[SerDes]]、[[DDR]] 等高速介面 IP 及基礎元件 IP 之研發設計與授權

**下游 (IC 設計公司與晶圓廠):**
- **晶圓代工夥伴:** [[台積電]] (最佳特殊製程 IP 合作夥伴)、[[聯華電子]] (UMC)
- **IC 設計客戶:** 全球 150+ 家 IC 設計公司，涵蓋 AI 晶片、網通、車用及消費性 IC 廠
