# 6423_億而得

## 業務簡介
**板塊:** Technology
**產業:** Semiconductors
**市值:** 3,436 百萬台幣
**企業價值:** 3,540 百萬台幣

億而得微電子股份有限公司 (6423，Yield Microelectronics) 是一家深度專注於嵌入式非揮發性記憶體 ([[eNVM]]) [[矽智財]] IP 授權的微電子公司，商業模式為純 IP 授權 (Royalty + NRE)。公司核心技術涵蓋單次可程式化 ([[OTP]]) 及多次可讀寫的 eNVM IP，可在不改變標準 [[CMOS]] 製程下，協助下游 IC 設計廠商在晶片中低成本整合資料儲存、安全金鑰保護及出廠校準等功能。其 IP 被廣泛整合於 [[MCU]]、[[PMIC]]、顯示驅動器及 IoT 連接晶片中，已授權並量產於 [[台積電]]、[[聯電]]、[[格芯]] 等主流晶圓代工平台，是全球 eNVM IP 領域的重要獨立供應商。

## 供應鏈位置
**上游 (開發工具):**
- **EDA 工具:** Synopsys、Cadence (電子設計自動化軟體)
- **IP 驗證平台:** 與各大晶圓代工廠共同建立 [[eNVM]] IP PDK 並驗證

**中游:**
- **eNVM IP 設計:** **億而得微電子** ([[OTP]] 及可讀寫 [[eNVM]] [[矽智財]] IP 之研發、設計及授權)

**下游 (IP 被授權方):**
- **[[MCU]] 設計廠:** 嵌入 [[OTP]]/[[eNVM]] IP 於微控制器晶片中
- **[[PMIC]] 設計廠:** 嵌入調校參數儲存 eNVM
- **晶圓代工廠:** [[台積電]]、[[聯電]]、[[格芯]] (作為製程平台合作夥伴)

## 主要客戶及供應商
### 主要客戶 (IP 被授權廠商)
- **[[MCU]] 設計廠:** [[盛群半導體]]、[[松翰科技]] 等台灣 MCU 廠商 (整合 [[OTP]]/[[eNVM]] IP)
- **[[PMIC]] 廠:** 各大電源管理 IC 設計公司 (採用 eNVM 儲存校準資料)
- **物聯網/連接 IC 廠:** Wi-Fi、藍牙等 IoT 晶片設計公司
- **晶圓代工平台:** [[台積電]]、[[聯電]] (作為 PDK 合作驗證夥伴)

### 主要供應商
- **EDA 軟體:** Synopsys、Cadence (IP 研發工具)
- **IP 驗證:** 各大晶圓代工廠技術平台
