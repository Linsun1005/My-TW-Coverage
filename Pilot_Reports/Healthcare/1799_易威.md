# 1799 - [[易威]]

## 業務簡介
**板塊:** Healthcare
**產業:** Drug Manufacturers - Specialty & Generic
**市值:** 5,025 百萬台幣
**企業價值:** 5,030 百萬台幣

易威股份有限公司為台灣中型學名藥及藥品原料製造商，以固形口服製劑學名藥生產為核心，提供台灣健保藥品市場處方學名藥，產品線涵蓋中樞神經系統、消化系統及心血管領域。公司通過 TFDA cGMP 認證，具備片劑、膠囊及粉末製劑的製造能力。年營收約新台幣 10 億元，以台灣醫療院所健保藥品通路為主要市場。

## 供應鏈位置
**上游 (原料與設備):**
- **原料藥 API:** 進口各治療領域學名藥原料藥（亞洲 API 廠商）
- **賦形劑:** 標準藥用賦形劑（澱粉、乳糖等）

**中游:**
- **易威:** 學名藥固形製劑生產（壓片/填膠）→品質管制→TFDA 藥證申請

**下游 (終端應用):**
- **台灣健保市場:** 處方學名藥銷售至台灣醫院及診所
- **台灣藥局:** OTC 藥品及保健品藥局通路

## 主要客戶及供應商
### 主要客戶
- **台灣醫院及診所:** 健保學名藥採購

### 主要供應商
- **原料藥:** 亞洲 API 廠商
- **賦形劑:** 台灣及進口藥用賦形劑廠商
