# 1795 - [[美時]]

## 業務簡介
**板塊:** Healthcare
**產業:** Drug Manufacturers - Specialty & Generic
**市值:** 58,869 百萬台幣
**企業價值:** 98,474 百萬台幣

美時化學製藥股份有限公司為台灣中型特殊學名藥廠，主力以緩控釋（extended-release）口服學名藥為產品差異化定位，具備薄膜衣技術、多顆粒緩控釋技術（pellet coating）等特殊製劑能力，通過美國 FDA 及歐盟 EMA cGMP 認可。美時積極申請美國 ANDA（學名藥申請）及歐洲學名藥批文，以緩控釋關鍵劑型技術切入高競爭壁壘的特殊學名藥市場，避免與一般即釋型學名藥直接競爭。年營收約新台幣 60 億元，出口比重高。

## 供應鏈位置
**上游 (原料與設備):**
- **原料藥 API:** 進口學名藥 API（以緩控釋藥物 API 為主）
- **緩釋賦形劑:** 羥丙甲纖維素（HPMC）、乙基纖維素等緩釋基材廠商
- **製劑設備:** 流化床造粒包衣設備（[[Glatt]]等德國藥機廠）

**中游:**
- **美時化學:** 多顆粒緩控釋製劑研發（Process Development）→FDA ANDA 申請→cGMP 生產；緩控釋學名藥美國歐洲出口

**下游 (終端應用):**
- **美國學名藥市場:** FDA ANDA 核准學名藥在美銷售（美國主力）
- **歐洲學名藥市場:** 歐盟 EMA 認可國家學名藥銷售
- **台灣健保市場:** 本地學名藥健保通路

## 主要客戶及供應商
### 主要客戶
- **美國藥品通路商:** McKesson、AmerisourceBergen 等美國學名藥批發商
- **台灣醫院:** 台灣健保學名藥採購

### 主要供應商
- **API:** 印度及中國 API 廠商（[[Sun Pharma]]、中國原料藥廠）
- **緩釋賦形劑:** [[Dow Chemical]]（HPMC）等藥用賦形劑廠商
- **設備:** [[Glatt]] (德國) 流化床包衣設備
