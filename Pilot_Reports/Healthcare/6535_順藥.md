# 6535 - [[順藥]]

## 業務簡介
**板塊:** Healthcare
**產業:** Biotechnology
**市值:** 20,980 百萬台幣
**企業價值:** 20,740 百萬台幣

順天醫藥生技股份有限公司為台灣中型學名藥暨特殊藥品廠，主力業務涵蓋口服固形學名藥生產及慢性病藥品行銷，通過 TFDA cGMP 認證，同時持有部分醫師處方特殊學名藥許可。順藥以台灣健保市場為主要業務，產品線涵蓋心血管及代謝疾病學名藥，並代理部分國際藥廠品牌在台灣的特許銷售權。年營收約新台幣 15 億元。

## 供應鏈位置
**上游 (原料與設備):**
- **API 原料藥:** 進口學名藥原料藥（亞洲 API 廠商）
- **賦形劑:** 藥用輔料（乳糖、微晶纖維素等）
- **代理藥品:** 國際藥廠台灣代理授權藥品

**中游:**
- **順藥:** 口服固形學名藥製造；TFDA 藥品許可管理；健保通路銷售；代理藥品行銷

**下游 (終端應用):**
- **台灣健保市場:** 台灣醫院及診所學名藥健保採購
- **台灣藥局:** OTC 及處方藥藥局通路

## 主要客戶及供應商
### 主要客戶
- **台灣醫院及診所:** 健保學名藥採購

### 主要供應商
- **API 原料藥:** 亞洲學名藥原料藥廠商
- **代理藥品原廠:** 國際藥廠台灣代理授權廠商
