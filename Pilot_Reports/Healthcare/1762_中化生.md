# 1762 - [[中化生]]

## 業務簡介
**板塊:** Healthcare
**產業:** Biotechnology
**市值:** 2,146 百萬台幣
**企業價值:** 2,734 百萬台幣

中化生命科學股份有限公司為台灣中型製藥及生技公司，旗下業務涵蓋學名藥製造（口服製劑）、保健食品及農業生技（植物生長調節劑）等多元業務。公司以固形口服製劑（錠劑、膠囊）生產為核心，通過 TFDA cGMP 認證，主力學名藥品涵蓋消化系統、心血管及過敏藥物。農業生技部門專注植物保護生物農藥（[[植物生長調節劑]]）及天然生物活性物質研發。年營收約新台幣 15 億元。

## 供應鏈位置
**上游 (原料與設備):**
- **原料藥:** 進口學名藥原料藥（亞洲 API 廠商）
- **賦形劑:** 藥用賦形劑（澱粉、乳糖等）
- **農業生技原料:** 植物萃取活性成份及發酵生技原料

**中游:**
- **中化生命科學:** 口服固形製劑製造（壓錠/封膠）；農業生技製品生產；保健食品 OEM

**下游 (終端應用):**
- **台灣健保市場:** 學名藥處方藥銷售
- **農業通路:** 台灣農業植物用生技產品經銷
- **保健品 OEM:** 台灣品牌業者保健品委託製造

## 主要客戶及供應商
### 主要客戶
- **台灣醫院及診所:** 健保學名藥採購
- **農業通路:** 台灣農藥及生物農藥經銷商

### 主要供應商
- **API原料藥:** 亞洲原料藥廠
- **賦形劑:** 台灣及進口藥用賦形劑廠商
