# 6838 - [[台新藥]]

## 業務簡介
**板塊:** Healthcare
**產業:** Biotechnology
**市值:** 4,107 百萬台幣
**企業價值:** 2,806 百萬台幣

台灣新藥股份有限公司為台灣生技新藥開發公司，以台灣本土自研新藥為定位，管線涵蓋腫瘤及慢性病治療領域候選藥物，透過授權合作及自主推進臨床試驗。公司以小分子新藥及特殊製劑改良為研發策略，目標縮短研發時程並降低開發風險。年營收以政府補助及技術合作為主，處於臨床開發階段。

## 供應鏈位置
**上游 (原料與設備):**
- **化學合成:** 候選藥物 API 有機化學合成（委外精細化工廠）
- **製劑委外:** 臨床試驗用製劑 cGMP 委製
- **CRO 服務:** 臨床試驗受託服務

**中游:**
- **台灣新藥:** 自研新藥 IP 持有及優化；IND/NDA 法規推進；臨床合作

**下游 (終端應用):**
- **台灣醫院臨床試驗:** 台灣醫學中心臨床試驗合作機構
- **全球授權（未來）:** 候選藥物全球授權

## 主要客戶及供應商
### 主要客戶
- **臨床試驗醫院:** 台灣醫學中心臨床試驗合作

### 主要供應商
- **化學合成廠:** 精細化工 API 合成廠商
- **製劑 CMO:** 臨床用藥物委外 cGMP 製造廠
- **CRO:** 臨床試驗受託研究機構
