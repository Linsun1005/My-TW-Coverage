# 1789 - [[神隆]]

## 業務簡介
**板塊:** Healthcare
**產業:** Drug Manufacturers - Specialty & Generic
**市值:** 16,092 百萬台幣
**企業價值:** 13,902 百萬台幣

台灣神隆股份有限公司為台灣最大的原料藥（API）合約製造商（CDMO），以[[神隆]]品牌在全球原料藥 CDMO 市場享有高知名度，為全球頂尖製藥集團提供關鍵原料藥的合約開發與製造服務。神隆具備特殊化學合成能力（含高活性化合物 HAPI、抗腫瘤藥物 API），通過美國 FDA、歐洲 EMA、日本 PMDA 多國主管機關 cGMP 查廠認可。主要服務全球 Big Pharma（[[Pfizer]]、[[AstraZeneca]]、[[MSD]] 等），為其原廠藥產品提供獨家或多家 API 供應。年營收約新台幣 130 億元。

## 供應鏈位置
**上游 (原料與設備):**
- **有機化學原料:** 各類有機合成前驅物及試藥（進口化工廠商）
- **催化劑:** 貴金屬催化劑（鈀、鉑及手性催化劑，進口精化工廠）
- **高活性設備:** 密閉式高活性化合物合成設備（[[HAPI]]製造防護隔離設備）

**中游:**
- **台灣神隆:** 多步驟有機化學合成開發（Process Chemistry R&D）→規模放大（kilo-lab→Pilot plant→Commercial）→cGMP API 生產；多國監管認可

**下游 (終端應用):**
- **全球 Big Pharma 原廠藥:** [[Pfizer]]、[[AstraZeneca]]、[[Eli Lilly]]、[[Novartis]]等原廠藥 API 獨家或首要供應商
- **學名藥廠:** 全球學名藥廠 API 供應

## 主要客戶及供應商
### 主要客戶
- **全球大型製藥公司:** [[Pfizer]]、[[AstraZeneca]]、[[MSD/Merck]]、[[Eli Lilly]] 等 Big Pharma 原廠藥 API 採購
- **學名藥廠:** 全球學名藥廠高品質 API 採購

### 主要供應商
- **有機化學前驅物:** 進口精細化工廠商（歐洲及亞洲化工廠）
- **貴金屬催化劑:** 鈀、鉑等手性催化劑進口廠商（[[Johnson Matthey]]等）
- **高活性設備:** 封閉式高活性化合物處理設備製造商
