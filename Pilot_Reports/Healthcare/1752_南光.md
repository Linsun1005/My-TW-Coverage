# 1752 - [[南光]]

## 業務簡介
**板塊:** Healthcare
**產業:** Medical Instruments & Supplies
**市值:** 3,535 百萬台幣
**企業價值:** 3,845 百萬台幣

南光化學製藥股份有限公司為台灣中型學名藥製藥廠，主力業務為注射劑型學名藥（靜脈輸液、小針劑）及口服製劑生產製造，通過 TFDA cGMP／PIC/S GMP 認證。南光在台灣注射劑市場具有多年深耕基礎，提供台灣醫療院所靜脈注射液及小針劑藥品，常見治療領域涵蓋抗生素注射劑、電解質點滴液及術後止痛針劑等。近年積極升級無菌製造廠房符合更嚴格的國際 cGMP 標準，並評估歐美注射劑出口市場。年營收約新台幣 25 億元。

## 供應鏈位置
**上游 (原料與設備):**
- **注射用原料藥:** 進口抗生素、止痛及電解質等注射用 API
- **注射用水:** 製藥級注射用水（WFI，廠廠自製）
- **無菌包裝材料:** 玻璃安瓿、點滴瓶（玻璃或軟袋）及橡皮塞

**中游:**
- **南光化學:** 無菌配製→過濾→安瓿熔封或軟袋充填→滅菌→揭示性檢查→包裝；PIC/S cGMP 認證廠

**下游 (終端應用):**
- **台灣醫院:** 台灣醫學中心及地區醫院輸液及針劑採購（健保市場）
- **診所:** 台灣診所靜脈點滴及注射劑耗材

## 主要客戶及供應商
### 主要客戶
- **台灣醫學中心及醫院:** 健保注射劑採購（靜脈輸液及小針劑）

### 主要供應商
- **注射用原料藥:** 印度及中國注射用 API 廠商
- **玻璃包材:** 注射用玻璃安瓿及瓶廠（德國 Schott、中國廠）
- **橡皮塞:** 藥用注射用橡皮塞廠商
