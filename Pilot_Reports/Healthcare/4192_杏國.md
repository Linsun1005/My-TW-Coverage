# 4192 - [[杏國]]

## 業務簡介
**板塊:** Healthcare
**產業:** Biotechnology
**市值:** 698 百萬台幣
**企業價值:** 525 百萬台幣

杏國新藥股份有限公司為台灣生技新藥公司，旗下核心管線為抗胰臟癌一線化療新劑型 [[SB05PC]]（白蛋白紫杉醇 PEGylated 脂質體製劑 nab-paclitaxel + gemcitabine 改良製劑）及 [[SQ1004]] 抗黴菌新藥，致力於腫瘤學及抗感染領域新藥研發。公司以改良型新藥（505(b)(2)）策略為主，降低研發風險，同時善用特殊製劑技術提升既有藥物療效。年營收以技術授權及政府補助為主。

## 供應鏈位置
**上游 (原料與設備):**
- **抗癌 API:** paclitaxel 及 gemcitabine 等抗癌藥原料（委外 cGMP API 廠）
- **製劑委外:** [[SB05PC]] 脂質體製劑委外 CDMO 廠生產
- **CRO 服務:** 臨床試驗受託研究機構

**中游:**
- **杏國:** [[SB05PC]] 等改良型抗癌製劑管線開發；法規申請（IND、NDA）；技術授權談判

**下游 (終端應用):**
- **胰臟癌患者（未來）:** [[SB05PC]] 胰臟癌一線化療
- **授權合作夥伴:** 全球製藥公司改良型製劑授權

## 主要客戶及供應商
### 主要客戶
- **臨床試驗機構:** 台灣醫學中心腫瘤科臨床試驗
- **潛在授權廠商:** 全球製藥公司[[SB05PC]]等管線授權

### 主要供應商
- **抗癌 API:** 抗腫瘤 API 廠商（paclitaxel 等）
- **CDMO 廠:** 脂質體製劑委外 cGMP 廠商
- **CRO:** 臨床試驗受託機構
