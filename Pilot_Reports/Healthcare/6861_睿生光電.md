# 6861 - [[睿生光電]]

## 業務簡介
**板塊:** Healthcare
**產業:** Medical Devices
**市值:** 10,774 百萬台幣
**企業價值:** 9,063 百萬台幣

睿生光電股份有限公司為台灣 CMOS 型數位 X 光影像感測元件（[[CMOS X光影像感測器]]）製造廠商，主力產品為牙科數位感測器（口腔內 X-ray sensor）及醫療用平板偵測器（FPD）相關感測元件，提供全球牙科設備廠商及醫療影像設備廠商 OEM 感測器供應。公司具備 CMOS 感測器晶片整合、閃爍體（CsI 碘化銫）塗覆及封裝製程，通過 FDA、CE 及 ISO 13485 認證。年營收約新台幣 15 億元，出口為主。

## 供應鏈位置
**上游 (原料與設備):**
- **CMOS 晶片:** 醫療 X 光 CMOS Image Sensor 晶圓（委外半導體廠代工）
- **閃爍體材料:** 碘化銫（CsI）X 光閃爍體薄膜塗覆（[[CMOS X光影像感測器]]關鍵層）
- **封裝材料:** CMOS 感測器 PCB 及防水封裝材料

**中游:**
- **睿生光電:** CMOS 晶片→CsI 閃爍體塗覆→感測器組裝封裝→FDA/CE 認證；牙科及醫療影像 OEM

**下游 (終端應用):**
- **全球牙科設備廠:** 全球牙科 X-ray 設備品牌廠商 OEM 數位感測器
- **醫療影像設備廠:** 醫療 FPD 平板偵測器應用

## 主要客戶及供應商
### 主要客戶
- **全球牙科設備品牌:** 歐美牙科 X-ray 設備廠商[[CMOS X光影像感測器]] OEM 採購
- **醫療影像設備廠:** 醫療平板 X 光偵測器 FPD 廠商

### 主要供應商
- **CMOS 晶圓代工:** 醫療影像 CMOS 感測器晶圓代工廠
- **CsI 閃爍體:** 碘化銫薄膜塗覆材料廠商
- **封裝材料:** PCB 及感測器防護封裝廠商
