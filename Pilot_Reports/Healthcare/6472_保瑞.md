# 6472 - [[保瑞]]

## 業務簡介
**板塊:** Healthcare
**產業:** Drug Manufacturers - Specialty & Generic
**市值:** 52,005 百萬台幣
**企業價值:** 67,374 百萬台幣

保瑞生技股份有限公司為台灣最大製藥合約製造業者（CMO/CDMO），以承接國際及台灣藥廠學名藥及特殊製劑委託製造服務為核心業務，具備固形口服、注射劑、乾粉吸入劑及緩控釋等多種劑型製造能力，通過 FDA cGMP、EMA 及 TFDA PIC/S GMP 認證。保瑞近年積極透過併購（[[Alvogen Taiwan]]、[[Bora Pharmaceuticals]] 等）擴大產能並整合客戶基礎，成為亞太地區重要製藥 CDMO 整合者。年營收約新台幣 200 億元，為台灣醫藥 CMO 龍頭。

## 供應鏈位置
**上游 (原料與設備):**
- **API 原料藥:** 進口各治療領域 API（客戶指定 API 廠商，或自主採購）
- **賦形劑:** 多元藥用賦形劑（客戶配方指定輔料廠商）
- **製劑設備:** 多劑型生產線（高速壓片機、注射充填機、DPI 吸入裝置製造）

**中游:**
- **保瑞:** 多劑型 cGMP 委託製造（固形/注射/吸入）；製程開發服務（CDMO）；FDA/EMA 查廠維護

**下游 (終端應用):**
- **全球藥廠客戶:** 歐美日藥廠委託製造（OEM 生產後貼客戶標籤）
- **台灣學名藥市場:** 部分自有學名藥健保台灣銷售

## 主要客戶及供應商
### 主要客戶
- **歐美製藥客戶:** 歐美藥廠學名藥委託製造（CMO）
- **台灣藥廠:** 台灣製藥公司委託製劑代工

### 主要供應商
- **API 廠商:** 印度、中國及歐洲原料藥廠（客戶指定及自主採購）
- **賦形劑廠商:** 歐美及台灣藥用輔料廠商
- **設備採購:** 歐洲藥機廠（壓片機、注射充填機）
