# 3054 - [[立萬利]]

## 業務簡介
**板塊:** Healthcare
**產業:** Drug Manufacturers - Specialty & Generic
**市值:** 5,726 百萬台幣
**企業價值:** 5,128 百萬台幣

立萬利生技股份有限公司為台灣中小型學名藥及保健食品廠，主要生產口服固形製劑（錠劑、膠囊）學名藥及健康食品，在台灣健保藥品市場供應慢性病及急性病學名處方藥。公司通過 TFDA cGMP 認證，產品線涵蓋心血管、代謝疾病及中樞神經等治療領域學名藥。規模屬台灣中小型學名藥廠，主要深耕台灣本地健保市場。年營收約新台幣 8 億元。

## 供應鏈位置
**上游 (原料與設備):**
- **原料藥 API:** 進口學名藥原料藥（印度、中國 API 廠商）
- **賦形劑:** 藥用賦形劑（乳糖、微晶纖維素等）

**中游:**
- **立萬利生技:** 固形製劑學名藥生產（壓錠/填膠）；TFDA 藥證管理；健保藥品通路銷售

**下游 (終端應用):**
- **台灣健保市場:** 台灣醫院及診所健保學名藥採購
- **藥局:** 台灣藥局 OTC 藥品通路

## 主要客戶及供應商
### 主要客戶
- **台灣醫院及診所:** 健保學名藥採購

### 主要供應商
- **原料藥:** 印度及中國 API 廠商
- **賦形劑:** 台灣及進口藥用賦形劑廠商
