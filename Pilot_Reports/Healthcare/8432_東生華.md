# 8432 - [[東生華]]

## 業務簡介
**板塊:** Healthcare
**產業:** Drug Manufacturers - Specialty & Generic
**市值:** 1,962 百萬台幣
**企業價值:** 1,327 百萬台幣

東生華製藥股份有限公司為台灣中大型學名藥廠，具備固形口服製劑（錠劑、膠囊）及注射劑多元劑型製造能力，通過 TFDA PIC/S GMP 及部分 FDA cGMP 認證，產品線涵蓋心血管、抗感染、神經科及腫瘤相關學名藥，並提供部分授權學名藥（authorized generic）在台灣銷售。東生華以高品質及多元化學名藥產品線在台灣健保市場佔有重要份額，並積極擴大海外（東南亞、中東）學名藥出口。年營收約新台幣 40 億元。

## 供應鏈位置
**上游 (原料與設備):**
- **API 原料藥:** 大量進口學名藥原料藥（印度、中國 API 廠商，包括抗癌及特殊藥品 API）
- **賦形劑:** 多元藥用賦形劑（歐美及台灣藥用輔料廠）
- **注射包材:** 注射劑玻璃安瓿及輸液袋

**中游:**
- **東生華:** 多劑型學名藥製造（固形/注射）；TFDA/FDA cGMP 品質體系；台灣健保全市場銷售

**下游 (終端應用):**
- **台灣醫療院所:** 台灣學名藥健保全市場（醫學中心至基層診所）
- **海外出口:** 東南亞及中東學名藥出口

## 主要客戶及供應商
### 主要客戶
- **台灣醫院及診所:** 台灣全市場健保學名藥採購
- **海外學名藥通路:** 東南亞及中東藥品進口商

### 主要供應商
- **API 原料藥:** 印度[[Sun Pharma]]、[[Dr. Reddy's]]等 API 廠商
- **賦形劑:** 歐美藥用賦形劑廠商（[[BASF]]、[[Roquette]]等）
- **注射包材:** 進口藥用玻璃安瓿及輸液袋廠商
