# 4169 - [[泰宗]]

## 業務簡介
**板塊:** Healthcare
**產業:** Biotechnology
**市值:** 11,094 百萬台幣
**企業價值:** 20,277 百萬台幣

泰宗生物科技股份有限公司為台灣學名藥製造廠，主力生產口服固形學名藥（錠劑、膠囊）及部分非處方藥（OTC），通過 TFDA cGMP 認證，供應台灣健保市場及藥局通路。產品治療領域涵蓋腸胃消化、呼吸系統及一般感冒過敏學名藥。年營收約新台幣 10 億元，為台灣中小型藥廠。

## 供應鏈位置
**上游 (原料與設備):**
- **原料藥 API:** 進口學名藥原料藥（亞洲 API 廠商）
- **賦形劑:** 標準藥用賦形劑（乳糖、澱粉等）

**中游:**
- **泰宗生技:** 學名藥固形製劑生產（壓錠/填膠）；TFDA 藥品許可管理；健保通路銷售

**下游 (終端應用):**
- **台灣健保市場:** 台灣醫院及診所學名藥採購
- **台灣藥局:** OTC 藥品藥局通路

## 主要客戶及供應商
### 主要客戶
- **台灣醫院及診所:** 健保學名藥採購
- **台灣藥局:** OTC 藥品通路

### 主要供應商
- **API 原料藥:** 亞洲學名藥原料藥廠商
- **賦形劑:** 台灣及進口藥用輔料廠商
