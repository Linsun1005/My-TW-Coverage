# 6875 - [[國邑*]]

## 業務簡介
**板塊:** Healthcare
**產業:** Biotechnology
**市值:** 4,873 百萬台幣
**企業價值:** 3,682 百萬台幣

國邑藥品科技股份有限公司為台灣吸入型藥物遞送（inhalation drug delivery）新藥公司，以霧化吸入製劑（[[霧化吸入劑 nebulizer]]）技術平台開發肺部藥物遞送新藥，聚焦肺動脈高壓（[[肺動脈高壓]]）及肺部罕見疾病治療，旗下[[OT-001]]（吸入型前列環素乙基酯）用於[[肺動脈高壓]] PAH 治療臨床開發。公司以精準藥物吸入機制改善現有靜脈注射療法依從性，提供更方便的吸入替代治療法。年營收以技術授權及政府補助為主。

## 供應鏈位置
**上游 (原料與設備):**
- **吸入製劑 API:** [[OT-001]] 前列環素乙基酯活性成分合成
- **霧化裝置:** 醫療專用[[霧化吸入劑 nebulizer]]裝置設計
- **製劑委外:** 肺動脈治療吸入製劑 cGMP 製造

**中游:**
- **國邑:** [[OT-001]] 吸入製劑 IP 持有；[[肺動脈高壓]] PAH 臨床試驗；法規申請

**下游 (終端應用):**
- **台灣肺科醫師:** 台灣肺動脈高壓患者吸入治療
- **全球授權（未來）:** [[肺動脈高壓]] PAH 吸入製劑技術授權

- **API 合成廠:** 前列環素類 API 有機合成廠商
