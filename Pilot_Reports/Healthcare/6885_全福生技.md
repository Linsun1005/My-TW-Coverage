# 6885 - [[全福生技]]

## 業務簡介
**板塊:** Healthcare
**產業:** Biotechnology
**市值:** 3,046 百萬台幣
**企業價值:** 2,013 百萬台幣

全福生技股份有限公司為台灣眼科新藥研發公司，以乾眼症（[[乾眼症]]）及眼部發炎治療新藥為核心管線，旗下[[BRM421]]（滴眼液，神經生長因子 NGF 衍生物peptide）用於神經性角膜炎及[[乾眼症]]治療，已進入 Phase II/III 臨床試驗，並與美國合作夥伴共同推進 FDA 申請。全福以台灣眼科新藥研發在全球眼科學術網絡中具有能見度。年營收以技術授權及臨床研究合約為主。

## 供應鏈位置
**上游 (原料與設備):**
- **肽類 API:** [[BRM421]] 神經生長因子衍生肽合成（委外肽合成廠）
- **滴眼液製劑:** 眼用滴眼液無菌製劑委外充填
- **CRO 服務:** 眼科臨床試驗 CRO 合作

**中游:**
- **全福生技:** [[BRM421]] 肽類 API IP；滴眼液製劑開發；眼科 Phase II/III 臨床；技術授權

**下游 (終端應用):**
- **[[乾眼症]]患者（未來）:** 嚴重[[乾眼症]]及神經性角膜炎滴眼液治療
- **全球眼科授權：** 國際眼科藥廠 BRM421 授權商業化

## 主要客戶及供應商
### 主要客戶
- **臨床試驗醫院:** 台灣及美國眼科臨床試驗合作機構
- **授權合作夥伴:** 全球眼科藥廠[[BRM421]]授權

### 主要供應商
- **肽合成廠:** 眼科肽類 API 固相合成廠商
- **製劑充填:** 眼用無菌滴眼液充填 cGMP 廠
- **眼科 CRO:** 眼科臨床試驗 CRO 機構
