# 4746 - [[台耀]]

## 業務簡介
**板塊:** Healthcare
**產業:** Drug Manufacturers - Specialty & Generic
**市值:** 6,157 百萬台幣
**企業價值:** 8,258 百萬台幣

台耀化學股份有限公司為台灣特殊原料藥（API）及精細化學品合成廠，具備有機化學合成能力（多步驟合成、非對稱合成、高活性化合物），為學名藥廠提供學名藥 API 及中間體，同時提供農化及工業用精細化學品。公司通過 TFDA cGMP 及部分 FDA DMF 登記，在台灣學名藥 API 自主化供應扮演角色。年營收約新台幣 20 億元。

## 供應鏈位置
**上游 (原料與設備):**
- **化學原料:** 有機化學合成起始物及試藥（進口歐洲及亞洲化工廠）
- **催化劑:** 有機合成催化劑（鈀催化劑等）
- **反應設備:** 化學合成反應釜（不鏽鋼及搪玻璃反應釜）

**中游:**
- **台耀化學:** 多步驟有機合成→中間體→最終 API 精製；TFDA cGMP；FDA DMF 登記

**下游 (終端應用):**
- **台灣學名藥廠:** 學名藥 API 本土製造採購
- **全球製劑廠:** FDA DMF 登記 API 出口全球製劑廠

## 主要客戶及供應商
### 主要客戶
- **台灣製藥廠:** 台灣學名藥廠 API 採購
- **全球製劑廠:** FDA DMF API 出口全球製劑廠

### 主要供應商
- **化學起始物:** 歐洲及亞洲有機化學原料廠商
- **催化劑:** 鈀等貴金屬有機合成催化劑廠商
