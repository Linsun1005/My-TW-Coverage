# 1734 - [[杏輝]]

## 業務簡介
**板塊:** Healthcare
**產業:** Biotechnology
**市值:** 6,020 百萬台幣
**企業價值:** 6,844 百萬台幣

杏輝天力藥品工業股份有限公司為台灣中型學名藥及健康食品製造廠，主要生產固形口服製劑（學名藥錠劑、膠囊）及健康食品，通過台灣 TFDA cGMP 認證。公司產品包含降壓藥、降脂藥及鎮痛消炎藥等學名藥品，另設有健康食品品牌，提供保健膠囊及錠劑 OEM 服務。杏輝近年積極拓展生物相容製劑（如[[褪黑激素]]、[[維生素 D3]]等保健原料活性成分）的新藥研發，並嘗試進軍台灣特殊藥品市場。年營收約新台幣 30 億元。

## 供應鏈位置
**上游 (原料與設備):**
- **原料藥 API:** 進口學名藥原料藥（印度、中國 API 廠）
- **賦形劑:** 乳糖、微晶纖維素、羥丙甲纖維素等藥用賦形劑
- **健康食品原料:** 進口維生素、礦物質、植物萃取物原料廠（歐美保健原料廠）

**中游:**
- **杏輝天力:** 學名藥製劑製造（壓錠/封膠）；健康食品保健品 cGMP 生產；OEM 委託服務

**下游 (終端應用):**
- **台灣健保市場:** 學名藥健保給付銷售
- **藥局保健品:** 台灣藥局及藥妝保健食品零售
- **OEM 客戶:** 台灣保健品品牌業者 OEM 委託

## 主要客戶及供應商
### 主要客戶
- **台灣醫院及診所:** 健保學名藥採購
- **保健品品牌商:** 台灣保健食品品牌 OEM 委託

### 主要供應商
- **原料藥:** 印度、中國原料藥廠
- **賦形劑:** 歐美及台灣藥用輔料廠商
- **健康食品原料:** 歐美保健原料廠 (DSM Nutritional Products 等)
