# 4130 - [[健亞]]

## 業務簡介
**板塊:** Healthcare
**產業:** Biotechnology
**市值:** 4,027 百萬台幣
**企業價值:** 2,932 百萬台幣

健亞生物科技股份有限公司為台灣中型特殊學名藥廠，以中樞神經系統（CNS）及睡眠相關學名藥（安眠藥、抗焦慮藥）為核心產品差異化，具備特殊管制藥品（第四級管制藥品）製造許可，通過 TFDA cGMP 認證。健亞另積極布局台灣緩控釋口服製劑技術，並代理國際醫藥品牌在台銷售。年營收約新台幣 25 億元，CNS 學名藥在台灣健保市場具有一定市佔。

## 供應鏈位置
**上游 (原料與設備):**
- **CNS API:** 進口中樞神經系統及安眠藥 API（管制藥品原料藥，歐洲 API 廠）
- **賦形劑:** 藥用標準賦形劑（乳糖、微晶纖維素等）
- **管制藥包材:** 安全防偽管制藥品包裝用材料

**中游:**
- **健亞生技:** CNS 管制學名藥固形製劑製造；TFDA 管制藥品許可管理；健保通路及代理藥品銷售

**下游 (終端應用):**
- **台灣精神科及神經內科:** CNS 安眠及抗焦慮學名藥健保採購
- **台灣診所:** 慢性病 CNS 學名藥診所採購

## 主要客戶及供應商
### 主要客戶
- **台灣精神醫療院所:** 台灣精神科及神經科醫院 CNS 學名藥採購
- **台灣一般診所:** 睡眠及抗焦慮管制藥品採購

### 主要供應商
- **CNS 管制 API:** 歐洲管制藥品原料藥廠商
- **賦形劑:** 台灣及進口藥用輔料廠商
- **代理原廠:** 國際藥廠台灣代理藥品品牌
