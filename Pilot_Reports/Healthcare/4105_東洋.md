# 4105 - [[東洋]]

## 業務簡介
**板塊:** Healthcare
**產業:** Drug Manufacturers - Specialty & Generic
**市值:** 17,928 百萬台幣
**企業價值:** 16,968 百萬台幣

台灣東洋藥品工業股份有限公司為台灣中大型製藥公司，以學名藥製造及創新藥開發並行，產品線涵蓋抗腫瘤藥物（含[[微脂體阿黴素]] Lipo-Dox 等脂質體製劑）、心血管學名藥及靜脈注射製劑。東洋旗下 [[Lipo-Dox]]（隱形微脂體阿黴素注射液）為台灣自主研發的微脂體學名藥，在台灣及海外（美國 ANDA 申請）具有市場地位。公司通過 TFDA PIC/S GMP 及 FDA cGMP 認證，具備無菌注射劑、微脂體製劑及固形口服製劑等多元劑型製造能力。年營收約新台幣 50 億元。

## 供應鏈位置
**上游 (原料與設備):**
- **原料藥 API:** 進口學名藥 API（抗腫瘤及心血管藥物 API，印度及歐洲 API 廠商）
- **脂質體原料:** 磷脂質（HSPC、DSPE-PEG）等微脂體製劑關鍵輔料（[[Lipoid]] 等廠商）
- **無菌包裝:** 注射液無菌玻璃瓶及橡皮塞廠商

**中游:**
- **台灣東洋:** 微脂體製劑研發（[[Lipo-Dox]] 脂質體阿黴素）；無菌注射劑生產；固形口服學名藥；FDA ANDA 申請

**下游 (終端應用):**
- **台灣醫學中心腫瘤科:** [[Lipo-Dox]] 卵巢癌及乳癌化療臨床應用
- **美國市場:** FDA ANDA 核准微脂體學名藥出口
- **台灣健保學名藥:** 心血管及一般學名藥健保市場

## 主要客戶及供應商
### 主要客戶
- **台灣醫學中心:** 台灣腫瘤科醫師 [[Lipo-Dox]] 化療用藥採購
- **美國藥品通路商:** 美國 ANDA 核准學名藥出口客戶
- **台灣醫院及診所:** 心血管學名藥健保採購

### 主要供應商
- **磷脂質原料:** [[Lipoid]]（德國）及進口磷脂質製造廠商
- **抗腫瘤 API:** 進口抗腫瘤原料藥廠商
- **無菌包材:** 注射用玻璃瓶及橡皮塞廠商
