# 4157 - [[太景*-KY]]

## 業務簡介
**板塊:** Healthcare
**產業:** Biotechnology
**市值:** 7,190 百萬台幣
**企業價值:** 6,141 百萬台幣

太景生物科技股份有限公司（太景-KY，開曼島登記上市）為台灣新藥研發公司，核心成就為新型喹諾酮類抗生素 [[奈諾沙星]]（nemonoxacin，商品名[[太捷信]]），已於台灣及中國大陸取得藥品許可，適應症為社區獲得性肺炎（CAP）。太景是台灣少數完成本土自研抗菌新藥且已在亞洲市場上市的生技公司，代表台灣抗感染藥物研發重要突破。公司持續與大陸合作伙伴合作推廣太捷信，並探索[[奈諾沙星]]在抗 MRSA 等耐藥菌感染治療領域的擴大適應症。年營收以技術授權金及中國大陸銷售授權分潤為主。

## 供應鏈位置
**上游 (原料與設備):**
- **合成 API:** [[奈諾沙星]] API 有機化學合成（委外 cGMP API 廠）
- **製劑委外:** 委託 cGMP 製劑廠生產[[太捷信]]口服膠囊及靜脈注射製劑
- **臨床研究:** 臨床研究組織（CRO）支援

**中游:**
- **太景-KY:** [[奈諾沙星]] IP 持有及授權；台灣及中國大陸藥證管理；合作伙伴授權銷售

**下游 (終端應用):**
- **台灣醫院感染科:** [[太捷信]] 台灣社區肺炎抗生素用藥
- **中國大陸醫院:** 中國大陸授權合作伙伴推廣社區肺炎治療

## 主要客戶及供應商
### 主要客戶
- **台灣感染科醫師:** 台灣醫院感染科及呼吸科[[太捷信]]採購
- **中國大陸合作伙伴:** 中國大陸授權推廣[[奈諾沙星]]的藥廠

### 主要供應商
- **API 合成廠:** 委外 cGMP 有機化學合成[[奈諾沙星]] API 廠商
- **製劑 CMO:** 口服及注射[[太捷信]]製劑委外 cGMP 廠
- **CRO:** 臨床試驗受託研究機構
