# 3705 - [[永信]]

## 業務簡介
**板塊:** Healthcare
**產業:** Drug Manufacturers - Specialty & Generic
**市值:** 14,597 百萬台幣
**企業價值:** 14,171 百萬台幣

永信藥品工業股份有限公司為台灣歷史悠久的學名藥大廠，創立於 1967 年，在台灣學名藥市場具有重要地位，產品線涵蓋慢性病學名藥（心血管、代謝、中樞神經）、抗生素及 OTC 成藥，全台醫院及診所市場均有廣泛覆蓋。永信具備固形製劑、液態製劑及注射劑等多元劑型製造能力，通過 TFDA PIC/S GMP 認證，並進行海外市場（東南亞、中東）學名藥出口。年營收約新台幣 70 億元，為台灣前五大學名藥廠之一。

## 供應鏈位置
**上游 (原料與設備):**
- **原料藥 API:** 大量進口學名藥 API（印度、中國及歐洲 API 廠商）
- **賦形劑:** 大量採購藥用賦形劑（歐美及台灣藥用輔料廠）
- **製劑設備:** 高速壓片機、自動化膠囊充填機及品管設備

**中游:**
- **永信藥品:** 多劑型學名藥製劑生產（錠劑/膠囊/液劑）；品管釋放→健保申請→台灣全市場醫院診所銷售

**下游 (終端應用):**
- **台灣醫院及診所:** 台灣健保學名藥全市場覆蓋（醫學中心至基層診所）
- **藥局:** 台灣藥局 OTC 藥品通路
- **出口:** 東南亞、中東學名藥出口

## 主要客戶及供應商
### 主要客戶
- **台灣醫療院所:** 台灣醫學中心、地區醫院及基層診所健保藥品採購
- **台灣藥局:** OTC 成藥藥局通路

### 主要供應商
- **原料藥:** 印度[[Sun Pharma]]、[[Aurobindo]]等 API 廠商
- **賦形劑:** 歐美藥用賦形劑廠商（[[BASF Pharma]]、[[Roquette]]等）
