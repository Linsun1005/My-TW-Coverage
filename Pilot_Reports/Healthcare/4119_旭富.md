# 4119 - [[旭富]]

## 業務簡介
**板塊:** Healthcare
**產業:** Drug Manufacturers - Specialty & Generic
**市值:** 5,766 百萬台幣
**企業價值:** 6,541 百萬台幣

旭富製藥科技股份有限公司為台灣原料藥（API）中間體及學名藥製造商，以腸胃科、鎮痛消炎及心血管等治療領域學名藥 API 中間體合成為核心，同時生產口服固形製劑學名藥供台灣健保市場。旭富具備有機合成化學能力，生產部分學名藥的關鍵合成中間體，並供應鏈下游製劑廠使用。通過 TFDA cGMP 認證，部分產品具備海外出口能力。年營收約新台幣 15 億元。

## 供應鏈位置
**上游 (原料與設備):**
- **化學原料:** 有機合成前驅物及化學試藥（進口精細化工廠商）
- **反應設備:** 化學合成反應釜及蒸餾設備
- **賦形劑:** 藥用賦形劑（固形口服製劑用）

**中游:**
- **旭富製藥:** 學名藥 API 中間體有機合成；固形口服製劑製造；TFDA cGMP 品管

**下游 (終端應用):**
- **台灣健保市場:** 固形口服學名藥健保採購
- **製劑廠 API 客戶:** 學名藥 API 中間體供應台灣及海外製劑廠

## 主要客戶及供應商
### 主要客戶
- **台灣製劑廠:** 台灣學名藥廠 API 中間體採購
- **台灣醫院及診所:** 健保學名藥採購

### 主要供應商
- **化學原料:** 進口精細化工有機合成原料廠商
- **賦形劑:** 台灣及進口藥用輔料廠商
