# 4173 - [[久裕]]

## 業務簡介
**板塊:** Healthcare
**產業:** Pharmaceutical Retailers
**市值:** 1,421 百萬台幣
**企業價值:** 1,327 百萬台幣

久裕興業股份有限公司為台灣農藥及植物保護藥劑製造商（屬 Healthcare 產業鏈邊界），主要生產農業用殺蟲劑、殺菌劑及除草劑等植物保護化學品，供應台灣及出口亞太農業市場。公司持有 TFDA 農藥登記許可，具備農藥製劑配製及分裝能力，以台灣本土農業通路及東南亞市場為主要客群。年營收約新台幣 15 億元。

## 供應鏈位置
**上游 (原料與設備):**
- **農藥 API（原體）:** 進口殺蟲劑及殺菌劑原體（中國及印度農藥原體廠）
- **助劑:** 乳化劑、分散劑等農藥製劑助劑
- **包裝材料:** 農藥專用 HDPE 瓶及鋁袋

**中游:**
- **久裕興業:** 農藥製劑調配（乳劑、水懸劑、粒劑）；農藥登記；台灣農業通路銷售；出口

**下游 (終端應用):**
- **台灣農業通路:** 台灣農藥經銷商及農民直銷
- **東南亞農業:** 東南亞農藥出口（水稻及蔬菜作物保護）

## 主要客戶及供應商
### 主要客戶
- **台灣農藥經銷商:** 台灣農業資材通路商
- **東南亞農業通路:** 東南亞農藥進口代理商

### 主要供應商
- **農藥原體:** 中國及印度農藥 API 原體廠商
- **助劑:** 農藥製劑乳化劑及分散劑化工廠商
- **包材:** 農藥用 HDPE 容器廠商
