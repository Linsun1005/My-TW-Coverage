# 3716 - [[中化控股]]

## 業務簡介
**板塊:** Healthcare
**產業:** Medical Distribution
**市值:** 4,720 百萬台幣
**企業價值:** 7,002 百萬台幣

中化生技控股股份有限公司為台灣[[中化集團]]旗下醫療配銷及健康照護集團控股公司，旗下涵蓋中化製藥（學名藥）、中化裕民健康事業（藥品醫材配銷）及多元健康照護事業，在台灣醫藥配銷及健康照護服務鏈具有完整佈局。中化集團以台灣藥品批發配銷（醫院及診所藥品供應鏈）及特色學名藥生產為核心，另在健康照護場域（連鎖藥局、長照等）積極布局。年營收約新台幣 200 億元（集團合計）。

## 供應鏈位置
**上游 (原料與設備):**
- **原廠藥:** 進口國際原廠藥代理（MSD、輝瑞等大型跨國藥廠台灣採購）
- **學名藥製造:** 旗下中化製藥自製學名藥 API 採購及生產
- **醫療器材:** 配銷醫療器材品牌進口商品

**中游:**
- **中化生技控股:** 旗下中化製藥學名藥製造；中化裕民藥品批發配銷（全台醫療院所供應鏈）；健康照護事業

**下游 (終端應用):**
- **台灣醫院及診所:** 藥品配銷至全台醫療院所（健保採購通路）
- **連鎖藥局:** 藥品批銷至台灣連鎖藥局

## 主要客戶及供應商
### 主要客戶
- **台灣醫院及診所:** 台灣全市場醫療院所藥品配銷
- **連鎖藥局:** 台灣連鎖藥局藥品批走供應

### 主要供應商
- **原廠藥廠:** [[Pfizer]]、[[MSD/Merck]]等跨國藥廠台灣代理進口
- **學名藥 API:** 亞洲原料藥廠商（旗下中化製藥用）
- **醫材品牌:** 各醫療器材品牌配銷進口廠商
