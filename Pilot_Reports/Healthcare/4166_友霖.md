# 4166 - [[友霖]]

## 業務簡介
**板塊:** Healthcare
**產業:** Drug Manufacturers - Specialty & Generic
**市值:** 6,331 百萬台幣
**企業價值:** 5,256 百萬台幣

友霖生技醫藥股份有限公司為台灣中型製藥公司，以固形口服製劑（錠劑、膠囊）學名藥及保健食品生產為核心，通過 TFDA cGMP 認證，供應台灣健保市場及保健食品 OEM 客戶。產品線涵蓋抗過敏、消化系統及心血管學名藥，另提供機能性保健食品委託製造服務。年營收約新台幣 15 億元，以台灣本地市場為主。

## 供應鏈位置
**上游 (原料與設備):**
- **原料藥 API:** 進口學名藥原料藥（亞洲 API 廠商）
- **賦形劑:** 標準藥用賦形劑（乳糖、微晶纖維素等）
- **保健原料:** 進口保健食品原料（維生素、礦物質等）

**中游:**
- **友霖:** 固形口服學名藥製造；保健食品 cGMP OEM；TFDA 藥品許可管理

**下游 (終端應用):**
- **台灣健保市場:** 學名藥健保採購醫院及診所通路
- **保健食品 OEM:** 台灣保健品品牌業者委託製造

## 主要客戶及供應商
### 主要客戶
- **台灣醫院及診所:** 健保學名藥採購
- **保健品品牌商:** 台灣保健食品 OEM 委製商

### 主要供應商
- **原料藥:** 亞洲 API 廠商
- **賦形劑:** 台灣及進口藥用輔料廠商
- **保健食品原料:** 進口維生素及植物萃取保健原料廠
