# 6672 - [[騰輝電子-KY]]

## 業務簡介
**板塊:** Technology
**產業:** Electronic Components
**市值:** 15,037 百萬台幣
**企業價值:** 13,026 百萬台幣

騰輝電子-KY (6672，Ventec International) 為專注利基型高階 [[PCB]] 相關材料製造商，核心產品包含散熱鋁基板 (IMS，全球車用 LED 頭燈散熱基板領導廠) 及高頻高速[[銅箔基板]] (CCL)。營收結構：CCL 約 36%、鋁基散熱板約 32%、膠片/半固化片約 20%。產品廣泛應用於軍工航太 (AS9100 認證，亞洲少數切入歐美高階國防供應鏈)、[[低軌衛星]]通訊 ([[PCB]]/天線板用高頻 CCL)、[[5G]] 基地台、[[電動車]]散熱及 [[AI 伺服器]]。軍工航太營收佔比預估達 30%。生產基地涵蓋中國蘇州/江陰/深圳及台灣，泰國新廠預計 2026 年投產。近年積極開發半導體先進封裝特殊薄膜材料及 tec-speed 系列高頻高速層壓板 (M9/M10)。

## 供應鏈位置

**上游 (基材原料):**
- **[[銅箔]]:** 高純度銅箔供應商
- **[[玻纖布]]:** 玻璃纖維布供應商
- **特殊樹脂:** 環氧樹脂、[[聚醯亞胺]] (PI)、PTFE 等高階特用化學樹脂
- **鋁板:** 金屬散熱基板用鋁材

**中游 (特殊 CCL/基板製造):**
- **CCL 製造:** **騰輝電子** — 高頻高速[[銅箔基板]] (tec-speed M9/M10)、軍工航太用 PI 材料、[[低軌衛星]]高頻 CCL
- **散熱基板:** **騰輝電子** — 車用 LED 散熱鋁基板 (IMS，全球領導)

**下游 (PCB 製造與終端):**
- **軍工航太:** 歐美國防 [[PCB]] 製造商 (飛彈系統、無人機、軍用飛機主板)
- **[[低軌衛星]]:** 衛星通訊 [[PCB]]/天線板製造商
- **車用照明 Tier 1:** [[Koito]] (小糸)、[[Stanley]] (斯坦利) — 車用 LED 頭燈散熱基板
- **[[AI 伺服器]]:** 高速 [[PCB]] 板廠
