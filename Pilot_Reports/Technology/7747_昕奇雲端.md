# 7747 - [[昕奇雲端]]

## 業務簡介
**板塊:** Technology
**產業:** Software - Infrastructure
**市值:** 3,593 百萬台幣
**企業價值:** 2,880 百萬台幣

昕奇雲端科技股份有限公司（GCS，GrandTech Cloud Sciences）為台灣多雲MSP及雲端FinOps管理服務商，以[[AWS]]、[[Google Cloud Platform]]及[[Microsoft Azure]]三大雲端平台的代理商及管理服務夥伴（MSP）身份，提供企業多雲架構諮詢、自主研發[[ARMIN]] FinOps成本最佳化平台（多雲即時費用監控）及雲端资安管理服務。公司具備[[ISO 27001]]資安認證及多雲架構能力，客戶為台灣及東南亞新創獨角獸及數位原生企業，已服務旅遊科技平台[[KKday]]、數位媒體軟體商[[Cyberlink]]、電商SaaS[[91APP]]及串流影音[[LINE TV]]等知名企業。2025年7月於OTC掛牌，年營收約新台幣15億元。

## 供應鏈位置
**上游 (原料與設備):**
- **雲端平台服務:** [[AWS]]、[[Google Cloud Platform]]、[[Microsoft Azure]]雲端運算資源（多雲代理）
- **資安技術:** [[ISO 27001]]認證雲端資安技術棧
- **FinOps工具:** 多雲成本管理API整合

**中游:**
- **昕奇雲端:** [[ARMIN]] FinOps多雲成本優化平台研發；多雲MSP架構服務；雲端資安管理；[[AWS]]/[[Google Cloud Platform]]/[[Microsoft Azure]]三雲代理銷售及諮詢

**下游 (終端應用):**
- **台灣數位原生企業:** 新創及網路公司多雲管理及FinOps服務
- **東南亞企業:** 東南亞雲端MSP服務擴展

## 主要客戶及供應商
### 主要客戶
- **旅遊及電商科技公司:** [[KKday]]（旅遊平台）、[[91APP]]（電商SaaS）多雲架構服務
- **數位媒體及串流平台:** [[Cyberlink]]（友立）、[[LINE TV]]多雲成本及基礎設施管理

### 主要供應商
- **雲端平台:** [[AWS]]（Amazon Web Services）、[[Google Cloud Platform]]、[[Microsoft Azure]]雲端資源代理
- **資安認證機構:** [[ISO 27001]]資安管理體系認證機構
