# 6910 - [[德鴻]]

## 業務簡介
**板塊:** Technology
**產業:** Software - Application
**市值:** 900 百萬台幣
**企業價值:** None 百萬台幣

德鴻科技股份有限公司（[[邁達特集團]]成員）為台灣客服中心（Contact Center）技術及AI語音分析解決方案商，主力產品為智能語音機器人、語音分析系統、全渠道客戶互動平台及企業應用中台軟體，服務金融保險、電信、物流、電商及遊戲等行業大型企業客戶。公司以[[Genesys]]技術合作夥伴資格切入全渠道客服中心部署，並深化AI語音互動與自然語言理解技術，連續獲得[[Gartner]]報告認可。2025年12月在台灣OTC掛牌，為[[邁達特集團]]旗下信息服務子公司。年營收約新台幣15億元。

## 供應鏈位置
**上游 (原料與設備):**
- **CC平台技術:** [[Genesys]]客服中心全渠道平台（OEM整合授權）
- **雲端基礎設施:** [[AWS]]、[[Microsoft Azure]]雲端聯絡中心基礎設施
- **AI語音技術:** 自然語言理解及語音辨識AI引擎

**中游:**
- **德鴻:** AI語音機器人+語音分析系統整合；[[Genesys]]全渠道聯絡中心部署；企業應用中台研發；企業大型客服系統維護

**下游 (終端應用):**
- **金融保險業:** 台灣及大中華金融保險業客服中心AI升級
- **電信、電商、遊戲業:** 大型企業全渠道客戶互動平台部署

## 主要客戶及供應商
### 主要客戶
- **台灣大型金融保險機構:** 金融業客服中心AI語音機器人及語音分析系統採購
- **台灣電信及電商業者:** 全渠道聯絡中心平台及中台整合服務

### 主要供應商
- **CC平台廠商:** [[Genesys]]（美國）全渠道客服中心平台授權
- **AI技術:** 自然語言理解及語音辨識AI引擎廠商
- **雲端平台:** [[AWS]]、[[Microsoft Azure]]聯絡中心雲端基礎設施
