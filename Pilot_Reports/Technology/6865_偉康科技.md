# 6865 - [[偉康科技]]

## 業務簡介
**板塊:** Technology
**產業:** Software - Infrastructure
**市值:** 563 百萬台幣
**企業價值:** None 百萬台幣

偉康科技股份有限公司（WebComm Technology）為台灣資訊軟體服務商，以自主開發的企業級軟體解決方案及資訊系統整合為主力，提供中小企業及政府機構的軟體應用服務。公司主要業務涵蓋企業資訊管理系統軟體授權、客製化軟體開發及技術支援服務。2023年1月在台灣OTC掛牌，近期積極透過私募引入策略性投資人推動垂直或水平整合。年營收約新台幣1億元，屬小型軟體服務公司。

## 供應鏈位置
**上游 (原料與設備):**
- **基礎設施:** 雲端服務平台（[[Microsoft Azure]]、[[AWS]]等）
- **開發工具:** 企業應用軟體開發框架
- **資料庫:** 企業應用資料庫系統（[[Microsoft SQL Server]]等）

**中游:**
- **偉康科技:** 企業資訊管理系統軟體研發；客製化系統整合；技術維護服務

**下游 (終端應用):**
- **台灣中小企業:** 企業資訊管理軟體及系統導入
- **台灣政府機關:** 資訊化應用系統採購

## 主要客戶及供應商
### 主要客戶
- **台灣中小企業:** 企業資訊管理系統軟體採購
- **台灣政府及公共機構:** 資訊軟體系統服務採購

### 主要供應商
- **雲端基礎設施:** [[Microsoft Azure]]、[[AWS]]雲端平台
- **資料庫軟體:** [[Microsoft SQL Server]]等企業應用資料庫廠商
- **開發工具:** 軟體開發框架及工具廠商
