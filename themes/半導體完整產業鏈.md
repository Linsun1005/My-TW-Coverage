# 半導體完整產業鏈 (台 / 美 / 日 / 韓 / 歐) — 投資決策版

> 從最上游電子級多晶矽、特殊氣體、光阻,到晶圓代工、先進封裝、IC 載板,再到下游 AI 伺服器的全鏈對照。
> **15 個主環節 × 64 個子類 × 3 級梯隊 (Tier) × 14 種主題標籤 × 5 類風險旗標**,為選股工具而非僅地圖。

## 使用方式

- **主環節 (`##`):** 15 個,依上下游切分,每檔只歸一個主環節。
- **子類 (`###`):** 主環節內按產品 / 技術 / 製程細分共 64 個。
- **Tier 分級 (每檔標示):**
  - **Tier 1 龍頭** — 市佔領先、定價權強、全球前列。
  - **Tier 2 第二梯隊** — 具規模但追隨者,享有產業 Beta。
  - **Tier 3 利基** — 小而專,題材彈性高,風險亦高。
- **主題標籤 (`#tag`):** 跨類橫向篩選,附錄提供完整索引。
- **風險旗標:** 🟢 防禦 / 🟠 景氣循環 / 🔵 陸廠紅鏈壓力 / 🔴 虧損或低毛利 / 🟣 小型題材。
- **子類同業對比表 (📊):** 市值 / P/E / 毛利率 / 核心賣點,方便跨檔快速比對。

**覆蓋統計:** 台股 250 檔 (Tier1 74 / Tier2 81 / Tier3 95) × 15 主環節 × 64 子類 × 14 主題標籤  |  國際對照 100+ 家龍頭

1. **主環節 (15 個)** — 依上下游位置切分,每檔只歸一個主環節。
2. **子類 (64 個)** — 主環節內再按產品/技術/製程細分,關鍵詞匹配為主,硬編碼為輔。
3. **Tier 梯隊 (3 級)** — 市值排序 + 龍頭硬編碼;同子類內 top 1-2 + 市值 300 億以上 → Tier 1;100 億以上或 top 1/2 → Tier 2;其餘 → Tier 3。

### 主題標籤規則

- **關鍵詞掃描**業務簡介 + 公司名稱自動打標。
- **龍頭硬編碼**:[[Apple]]/[[NVIDIA]] 供應鏈、CSP ASIC、中國去美化等較難從文字判斷之主題,以 ticker 白名單強制打標。
- 每檔可同時具有多個主題 (如 [[台積電]] 同時 `#CoWoS #HBM #AI伺服器 #車用半導體 #Apple供應鏈 #NVIDIA供應鏈 #CSP自研ASIC #先進封裝 #SiC第三代 #中國去美化`)。

### 風險標籤規則

| 旗標 | 觸發條件 |
|---|---|
| 🟢 龍頭防禦 | 市值 ≥ 1 兆 且 毛利率 ≥ 40% |
| 🟢 穩健獲利 | 市值 ≥ 1,000 億 且 淨利率 ≥ 15% 且 毛利率 ≥ 25% |
| 🟠 景氣循環 | 子類屬記憶體 / DDI / LED |
| 🔵 陸廠紅鏈壓力 | 子類屬成熟製程 / 8 吋 / MOSFET / Diode / 傳統封裝 |
| 🔴 當前虧損 | 最近年度淨利率 < 0 |
| 🔴 低毛利 | 最近年度毛利率 < 15% |
| 🟣 小型題材 | 市值 < 100 億 |
